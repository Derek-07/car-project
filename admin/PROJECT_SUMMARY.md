## 🎨 Customization Guide

### Change Colors
Edit `:root` variables in `admin-style-premium.css`:
```css
:root {
    --primary: #d4af37;           /* Gold */
    --dark-bg: #0f0f0f;           /* Dark background */
    --sidebar-bg: #1a1a1a;        /* Sidebar */
    --success: #48bb78;           /* Success color */
    --danger: #ff6b6b;            /* Danger color */
}
```

### Modify Navigation
Edit the `<nav class="sidebar-nav">` section in any HTML file.

### Add New Pages
1. Copy existing page HTML
2. Change title & breadcrumb
3. Update page content
4. Add navigation link

### Change Text/Labels
Directly edit text in HTML files.
